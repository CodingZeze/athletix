# AthletiX - Sports Analytics Platform

A modern sports analytics platform built with Next.js.

## Quick Start

### Local Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
